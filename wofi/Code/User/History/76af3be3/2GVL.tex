Objetivos Gerais
O objetivo geral do trabalho é desenvolver um back-end para um sistema multiplataforma para a empresa júnior do curso de Bacharelado em Engenharia de Computação da UEMA.

Objetivos Específicos
Pesquisar sobre as principais tecnologias utilizadas na construção de um back-end para um sistema multiplataforma de acordo com os requisitos da empresa júnior;
Analisar as melhores ferramentas para uso no desenvolvimento do back-end do sistema multiplataforma da empresa júnior;
Implementar as melhores soluções de acordo com as necessidades e requisitos da empresa júnior;
Avaliar o protótipo com a integração do front-end desenvolvido, por meio de teste de software e do uso em um sistema real;
Produzir um capítulo de livro a partir dos trabalhos realizados para submissão e possível publicação.
