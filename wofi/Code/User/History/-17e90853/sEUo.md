# Documentação

Esta pasta contém a documentação de desenvolvimento e arquitetura do projeto. Os principais documentos são:

- `overview.md` - Breve visão geral do projeto e suas restrições.
- `conventions.md` - Convenções de nomenclatura e organização.
- `architecture.md` - Arquitetura de alto nível do sistema e divisão de componentes.
- `how-it-works.md` - Detalhes de implementação e pipeline de renderização.
- `deployment.md` - Instruções de build e deployment.
- `developer-guide.md` - Passos práticos para estender e testar o projeto.
